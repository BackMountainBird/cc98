UserScripts for CC98

This repository contains some useful user scripts for cc98.org, which is a popular forum in Zhejiang University. All scripts support both Google Chrome and Mozilla Firefox with Scriptish or Greasemonkey installed.

Download

You can download the code of this project in either zip or tar formats. You can also clone the project with Git by running: 

$ git clone git://github.com/dangoakachan/cc98

Install Steps

If you are using Mozilla Firefox, you first need to install Greasemonkey or Scriptish extension. I'm now using Scriptish to develop this scripts, so the latter one is more recommended. Because Google Chrome natively supports Greasemonkey, you don't have to install an extra extension to use this scripts.

The installation of user scripts in both two browsers is very easy, just download the user script and drag it to the browser. A confirm window will pop up, make sure you want to install and click the install button.
